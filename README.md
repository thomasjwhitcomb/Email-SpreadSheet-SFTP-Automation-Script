# Email-Spreadsheet-SFTP Python Script
### A client-sponsored project for CSC131 @ Sacramento State University
#### The python script is built to scrape data from a user's email, build a spreadsheet of the desired data, and upload the data to a specified SSH server using SFTP.

### SFTP Script Notes:
- You must install the **paramiko** and **keyring** libraries, as the script has dependencies in each.
- An example of the Powershell command to run the script is present in the script header notes
